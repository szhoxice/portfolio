# Портфолио — Давид Миралиев

Статический сайт-портфолио (HTML, CSS, JS): главная, обо мне, проекты, навыки, контакты.

## Структура

- `html/` — страницы
- `css/`, `js/` — стили и скрипты
- `assets/images/` — изображения
- корневой `index.html` — переход на `html/index.html`

## Локально

Открой `index.html` или `html/index.html` в браузере; удобнее через локальный сервер (Live Server).

## GitHub Pages (по желанию)

В репозитории: **Settings → Pages → Source: Deploy from a branch → Branch `main` / folder `/ (root)`** — сайт откроется по адресу вида `https://szhoxice.github.io/portfolio/` (главная с корня сделает редирект на `html/index.html`).
